| 搜索 | 关键字输入 | 输入关键字进行搜索 |  |
|  | 行业选择 | 选择自己所在的行业 |  |
|  | 职位选择 | 选择自己想找的职业 |  |
|  | 期望工作地点 | 选择期望的工作地点 |  |
|  | 搜索历史 | 展示用户的搜索历史 |  |
| 直约职位 | 直约职位列表 |  |  |
|  | 直约职位详情 |  |  |
|  | 投递直约职位 |  |  |
| 推荐职位 | 推荐职位列表 |  |  |
|  | 推荐职位详情 |  |  |
|  | 投递推荐职位 |  |  |
| 职位列表 | 修改条件 |  |  |
|  | 查看已选内容 |  |  |
